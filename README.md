## ğŸ“Œ Project Overview

**AGRISENSE APP** is a UX case study focused on designing a **Smart Agriculture IoT mobile application** that helps farmers monitor field conditions, automate irrigation, and make data-driven decisions using AI-powered insights.

The primary goal of this project is to convert **complex IoT sensor data** into **simple, visual, and actionable information** that rural farmers can easily understand and trust.

This project was created as part of the academic subject **MDM â€“ Introduction to UI/UX**.

- Deliver **AI-based insights** instead of raw sensor numbers
- Design a **simple, intuitive, and multilingual UI**

## ğŸ” UX Research & User Insights

### Research Methods
- In-person interviews  
- WhatsApp voice calls  

### Key Findings
- Farmers prefer **large visuals over complex graphs**
- **Color-coded alerts** are easier to understand
- **Language support (Hindi / Marathi)** is very important
- Farmers do not trust raw numbers unless the **app explains what action to take**

### User Quotes
> â€œNumbers are confusingâ€¦ tell me what to do.â€  
> â€œIf moisture is low, the app should warn me early.â€  
> â€œGraphs should be simple.â€
